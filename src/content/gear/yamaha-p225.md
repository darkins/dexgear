## Keyboard Action

The P-225 uses **GHC (Graded Hammer Compact)** key action across all 88 keys:

- **Heavier resistance in the lower register**, progressively lighter toward the upper keys — mirrors the physical behaviour of an acoustic grand piano hammer mechanism.
- Designed for fast key repetition — the key mechanism resets quickly for repeated notes, essential for classical repertoire and fast pop piano lines.
- Not weighted to the same degree as Yamaha's GH3X action in higher-end models, but substantially heavier and more expressive than the standard semi-weighted keys common in entry-level keyboards.

## Main Voice

The P-225's flagship voice is the **CFX Concert Grand** — a sample set derived from Yamaha's flagship 9-foot concert grand piano used in international competitions:

- **Virtual Resonance Modeling (VRM) Lite:** simulates the sympathetic resonance between strings that occurs on an acoustic grand — notes held with the damper pedal resonate in character with other notes being played, dynamically shifting based on what's depressed.
- **88-key sampling:** each key is sampled individually at multiple velocity layers rather than using pitch-stretched samples.
- No tuning, voicing, or maintenance required.

## Built-in Features

- **71 preset songs** — classical, pop, and exercises for practice and listening reference.
- **Built-in metronome** with variable tempo and time signature.
- **10 rhythm patterns** for accompaniment and practice groove.
- **Dual-track internal recorder** — record left and right hand parts separately, then play them back together for part-checking during practice.

## MIDI Integration

The P-225 is a **class-compliant USB-MIDI device** — no driver required on Windows or macOS:

1. Connect via **USB-B to USB-A/C** cable to the computer.
2. Ableton automatically detects the P-225 as a MIDI input device.
3. Create a MIDI track and set its MIDI From to **Yamaha P-225** (or whichever name the OS assigns).
4. Load any virtual instrument on the track — the GHC velocity sensitivity delivers natural dynamic expression through the virtual instrument.

**5-pin MIDI output** is also available for connecting to the MOTU MIDI In when USB is used for another purpose (e.g., charging or audio recording via a separate interface).

## Using as a Soft Synth Controller in Ableton

The P-225's GHC action is well-suited as a primary piano controller for virtual instruments:

- **NI Noire** (Native Instruments): another CFX-based piano sample library — playing a CFX virtual through a CFX-sampled virtual creates a consistent piano feel.
- **Addictive Keys Studio Grand** (XLN Audio): detailed velocity layers and pedal resonance modelling that responds to the GHC's wide dynamic range.
- **Ableton Electric:** the on-board electric piano (Rhodes, Wurlitzer, DX-style) responds well to the P-225's velocity dynamics.

## Layering with the Hydrasynth Explorer

Create rich, evolving piano textures by layering the P-225 MIDI with the Hydrasynth:

1. Route P-225 USB MIDI to the same Ableton MIDI track as the Hydrasynth (or use two tracks with the same MIDI input channel).
2. Set the Hydrasynth to a slow-attack pad preset (long Attack, slow filter opening).
3. Play chords on the P-225 — the piano attack is immediate while the Hydrasynth pad swells underneath over 2–4 seconds, creating a natural dynamic layering.
4. Use the Hydrasynth's polyphonic aftertouch to add per-note filter swells based on P-225 note velocity routing in the Hydrasynth modulation matrix.

## Practice vs Production Workflow

<ul class="space-y-4 list-disc pl-6">
  <li><strong>Standalone practice:</strong> connect headphones to the P-225's front headphone jack. Access all 24 voices, built-in metronome, and rhythm patterns with no computer or studio gear powered on. Ideal for late-night practice or focused technique work.</li>
  <li><strong>Recording session:</strong> connect USB-B to the computer. Ableton sees the P-225 immediately as a MIDI controller. Use the Internal Recorder for quick sketches or load virtual instruments in Live for production-quality sound.</li>
  <li><strong>Dual use:</strong> keep the P-225's built-in sound off (Master Volume to 0 on the unit) while routing MIDI to Ableton — the GHC keybed becomes a pure controller with zero latency key response, all sound coming from the DAW through the MOTU and HS7 monitors.</li>
</ul>
