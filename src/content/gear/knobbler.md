## Features

- Knobbler provides a tablet-friendly control surface for Ableton Live with a number of workflow-focused features:

<ul class="space-y-4 list-disc pl-6">
  <li>High-resolution knobs (500 steps) with low-latency multitouch control.</li>
  <li>Knob state indicators: green = mapped or controlled elsewhere, red = automation recorded, grey = automation overridden (use Re-Enable Automation to restore).</li>
  <li>Two fast mapping modes: tap a selected parameter to map, or map directly from the Bluhand page; mappings are saved with the Live Set.</li>
  <li>Double-tap any slider to reset the parameter to its default value.</li>
  <li>Link adjacent sliders into X–Y pads for 2D control; split them back via the toolbar.</li>
  <li>Device Shortcuts: eight shortcut buttons to jump to important devices; map/unmap these for quick access.</li>
  <li>Bluhand page: parameter-focused view with parameter banks and Variations support for racks, plus fast mapping to Knobbler.</li>
  <li>Channel Strip mixer: access sends, volume, pan, mute, solo, record arm, crossfader assignment, and toggle record-enable (long press).</li>
  <li>Toolbar utilities: show/hide mixer, refresh display, unmapping mode, tempo and metronome, transport controls, capture MIDI, session automation overdub, loop, locators, and more.</li>
  <li>External hardware support: can work with external controllers (see `MIDI CC Bridge` for a workaround with Live 12 CC Control issues).</li>
  <li>Setup page: host discovery (or manual IP/port), version checks, and guidance for updating the Max for Live device.</li>
</ul>

- For full details, screenshots, and a feature walkthrough video, see the upstream documentation: https://plugins.steinkamp.us/m4l-Knobbler4/features
