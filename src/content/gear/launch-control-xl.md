## Hardware Layout

<ul class="space-y-4 list-disc pl-6">
  <li><strong>8 faders</strong> — control track volume by default in Ableton's Live mode.</li>
  <li><strong>8 top-row knobs (Send A)</strong> and <strong>8 middle-row knobs (Send B)</strong> — control the first two send amounts per track; pressing the Send Select button cycles to further sends.</li>
  <li><strong>24 RGB pads</strong> arranged in a 3×8 grid — default rows are: Mute (row 1), Solo (row 2), Record Arm (row 3).</li>
  <li><strong>8 track-select buttons</strong> — navigate the selected track bank left and right on the controller.</li>
  <li><strong>4 scene launch buttons</strong> — fire the next four scenes in Live's Session View.</li>
</ul>

## Templates

The LCX ships with 8 factory templates and supports 8 user-programmable templates. Hold **Template** and press one of the 8 matching pads to switch instantly between templates during a session. The active template name appears briefly in the LED ring response on the knobs.

## Ableton Live Integration

<ul class="space-y-4 list-disc pl-6">
  <li>Ableton recognises the LCX as a dedicated control surface. Set it in Preferences → Surfaces → Launch Control XL.</li>
  <li><strong>Send A/B Modes:</strong> the two rows of knobs default to Send A and Send B — ideal for Reverb and Delay return levels. Press the Send Select button to cycle through all available returns.</li>
  <li><strong>Mute/Solo/Arm grid:</strong> the three pad rows provide instant mute, solo, and record arm toggling across 8 tracks simultaneously — the fastest way to manage a large session without a mouse.</li>
  <li><strong>Device Mode:</strong> hold the Device button to temporarily map the top two knob rows to the parameters of the currently selected Live device (8 + 8 = 16 parameters visible at once).</li>
</ul>

## Complementing Push 1

The LCX and Push 1 are designed to co-exist on separate USB MIDI ports with zero MIDI conflict. The ideal split of responsibilities:

- **Launch Control XL:** volume faders, send A (reverb) and send B (delay) knobs, mute/solo/arm pads — the full mixing surface.
- **Push 1:** note input, step sequencing, clip launching, per-device parameter control — the performance instrument.

Keep the LCX in your peripheral vision while your hands are on Push for a complete DAW control surface without any reassignment needed.

## Integration with Knobbler

- **LCX layer:** faders and knobs handle the mixing layer (volume, sends).
- **Knobbler layer:** the iPad surface handles the device parameter layer (per-device encoder banks with auto-labels and X–Y pads).
- Keep both surfaces active simultaneously — LCX for mixing structure, Knobbler for sound design — providing a professional two-surface workflow without a hardware mixer.

## Template Customisation

Use **Novation Components** (novationmusic.com/components) to reprogram any knob, fader, or pad:

- Reassign MIDI CCs to any target (e.g. MIDI CC for Hydrasynth macro parameters)
- Change MIDI channels per control for routing to different instruments
- Save up to 8 custom user templates directly to the LCX flash memory

**Practical template suggestions:**

- **Template 1:** Live mixing defaults (Send A = Reverb return, Send B = Delay return)
- **Template 2:** Hydrasynth macro control (knobs → Hydrasynth macro CCs via MOTU MIDI Out)
- **Template 3:** Custom drum machine layout (pads → dedicated notes/CCs for hardware triggering)
