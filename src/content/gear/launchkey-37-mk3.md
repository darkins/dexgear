## Ableton Live Integration

The Launchkey 37 Mk3 is a tightly integrated Ableton control surface:

<ul class="space-y-4 list-disc pl-6">
  <li><strong>Capture MIDI:</strong> one button press captures the last few bars of anything played — no pre-record arming needed.</li>
  <li><strong>Transport controls:</strong> Play, Stop, Record, Loop, Click (metronome) directly on the hardware.</li>
  <li><strong>Track arm, clip launch, and scene launch</strong> all accessible from the pad grid without touching the mouse.</li>
  <li><strong>Quantise:</strong> instant quantise button applies to the currently focused clip.</li>
  <li><strong>Direct pad-to-clip mapping:</strong> the 16 pads map to a 4×4 section of the Session View clip grid; navigate up/down with the scene launch buttons.</li>
</ul>

## Scale Mode

Scale Mode is one of the Launchkey's most powerful live performance features:

- Transposes **all keys and pads** globally so only notes within the selected scale are output.
- Functions **standalone without a computer** — the filtering happens in the device's firmware.
- Set the root key and scale type directly on the hardware display.
- **Practical use:** match the Launchkey's scale and root key to the Hydrasynth patch's tuning. Every note from the Launchkey is guaranteed in-key, enabling jam sessions without wrong notes even when improvising freely.

## Chord Modes

Three different Chord Modes build full chords from a single key press:

- **Chord Mode 1 (Scale Chords):** plays triads and extended chords built from the current scale — major chords in major keys, minor chords in minor keys, automatically.
- **Chord Mode 2 (Fixed Chords):** defines a static chord shape that transposes chromatically with each key — useful for power chords or stacked intervals.
- **Custom Chords:** store up to 8 custom chord voicings and trigger them from the pads.

Chord Modes are ideal for building harmonic beds while Push 1 handles drum step sequencing.

## 4 Custom Modes

Configure up to 4 fully custom control layouts using **Novation Components** (browser-based editor):

- Assign any encoder or pad to any MIDI CC, Note, or Program Change message.
- Map to Hydrasynth macro CCs (e.g. CC 22–29 for Macros 1–8) — the Launchkey displays the CC number or a custom name on its screen.
- Map to VT-3 parameters (Pitch, Formant, Reverb, Character select via Program Change).
- Map to Ableton parameters using standard MIDI mapping.

## Standalone Hardware Control

Disconnect from the computer entirely and use the Launchkey 37's 5-pin MIDI output for hardware-only sessions:

- **Launchkey 37 MIDI Out → Hydrasynth MIDI In:** play the Hydrasynth directly with Scale Mode and Chord Modes active. Full expressive performance with no computer required.
- **Launchkey 37 MIDI Out → MOTU MIDI In:** route to the MOTU to forward to other hardware via MOTU's MIDI Out, or for use when recording to a standalone recorder.

## Integration Tips

<ul class="space-y-4 list-disc pl-6">
  <li><strong>Keys layer with Push 1:</strong> keep both active simultaneously in Ableton. Push handles drum step sequencing; Launchkey handles melodic keyboard input on a separate MIDI track. Two performers' worth of control on one machine.</li>
  <li><strong>Custom Mode for synth editing:</strong> map the 8 encoders to the Hydrasynth's 8 Macro CCs. While playing the Hydrasynth from the keys, the encoders provide real-time macro control in the same hand position.</li>
  <li><strong>Scale Mode + Hydrasynth:</strong> set both the Launchkey Scale Mode and the Hydrasynth's built-in scale (if enabled) to the same root key for guaranteed in-key performance across both controllers.</li>
  <li><strong>NKS compatibility:</strong> works seamlessly with Native Instruments instruments (Kontakt, Massive X) — the Launchkey display shows parameter names from NKS-tagged presets.</li>
</ul>
